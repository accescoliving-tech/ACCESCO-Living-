# ACCESCO Card Match

A fun memory card matching game with an ACCESCO theme!

## How to Play

1. Click "Start Game" to begin
2. Match pairs of ACCESCO-themed cards by clicking on them
3. You have 2 minutes to match all 6 pairs
4. Correct matches give you points and bonus time
5. Wrong matches cost you time
6. Try to get the highest score!

## Features

- 12 cards with ACCESCO-themed icons (food, fashion, shopping, services)
- Timer with time bonuses for correct matches
- Score system with leaderboard
- Responsive design that works on mobile and desktop
- Local storage for saving high scores

## Running the Game

Simply open `index.html` in any modern web browser. No server required!

## Technologies Used

- Vanilla JavaScript
- HTML5
- CSS3 with animations
